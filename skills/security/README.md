# Security

Fork-owned security engineering bucket. It is intentionally outside the
`mattpocock/skills` synchronization boundary, keeping the two upstreams
conflict-free.

- [`security-audit`](security-audit/SKILL.md) — Synchronized from
  [`cloudflare/security-audit-skill`](https://github.com/cloudflare/security-audit-skill)
  through a review-only pull request. Audit applications, APIs,
  services, CLI tools, libraries, daemons, and binaries for exploitable security
  issues with validated evidence and structured reporting.
