# Security

Fork-owned security engineering skills. This bucket is installed by the
Antigravity adapter but is intentionally absent from the upstream Claude plugin
manifest, keeping upstream-owned files conflict-free.

- [`security-audit`](security-audit/SKILL.md) — Audit applications, APIs,
  services, CLI tools, libraries, daemons, and binaries for exploitable security
  issues with validated evidence and structured reporting.
